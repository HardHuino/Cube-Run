# Cube-Run
Projet d'informatique graphique, Polytech-Université Paris-Saclay 2026

# ATTENTION : INSTALLATION

FR : Si vous ne souhaitez pas utiliser la version pré-build, il faudra build le dossier cube/ avec CMAKE, générer le projet via VS 2022 et mettre les dossier shaders/ et textures/ dans le Debug/ créé pour que le projet fonctionne.

EN : If you do not want to/can't use the pre-built version, you will need to build the cube/ folder with CMAKE, generate the project via VS 2022, and place the shaders/ and textures/ folders in the created Debug/ for the project to work.
